Setup Instructions:
Setting B.I.A.B. is pretty easy.

1- First of all, you need to be a Bravenet Affiliate, if not yet, please Click Here to signup and register yourself as a new afiliate ( click "Register for the Affiliate Program" after login)

2- This package comes in a zip format, so just unzip it to your local hard disk.
A new folder called "webtools" will be unzipped containing the necessary files to be edited and uploaded.

3- Within the "webtools" folder, find a file called "config.php". Open it using a text editor, like notepad, and change the $aff value to match your own Bravenet´s Affiliate ID.
For example, if your affiliate code given is something like: http://www.bravenet.com/cserv.php?pid=4&tid=2&afilid=123456789 means that your affiliate ID is: 123456789.
Then, put this "123456789" value in the config.php and save it.

It should look like
$aff = "123456789";     Dont forget the double quotes and the semicolon at the end of the line, or it will not work properly.
That´s it

4- Last step is to upload the full package to your web server using a FTP program keeping the file structure to a new folder, say "webtools", in your root directory. After uploaded, just point your browser to http://www.yourdomain.com/webtools and you are ready to go.
Note: Be sure to upload all the files in the "images" folder in binary format, and all *.php and *.html inASCII

5- The pack comes with an easy to edit header and footer files that will help you to match your current layout.
Just edit header.html and footer.html with your favorite HTML editor and replace the ones in your server.
If you want to change the hyperlinks colors (green by default) just open header.html and replace the styles color values to mach your site.
This pack also comes with a custom thankyou.php page that is launched after the signup form submitted. Feel free to edit this page as you want but be sure to save it in your server with the exact "thankyou.php" filename or it will fail to load after submitted. Anyway, you can always redirect this page to another one by adding a meta refresh parameter after  the <head> tag, like this:

<?php
include("config.php");
include("header.html");
?>
<html>
<head>
<meta http-equiv="refresh" content="0;url=http://www.yahoo.com">
</head>
<body>
</body>
</html>

<?php
include("footer.html");
?>

Pasting the above code to replace the current one located at the "thankyou.php" file will make the form to reach the thankyou page , then it will automatically redirects the visitor to yahoo.com (change the yahoo´s URL with yours, of course)

You are all set.
