1. Open up the "db.php" file, enter your MySQL details.

2. Open up "admin/index.php" and change the default admin user/pass

3. Upload all the files in the "upload" directory to the location you wish to install the system into

4. Run the installer by going to "http://www.yoursite.com/articles/install"

5. After running the installer, MAKE SURE YOU DELETE THE INSTALL FOLDER. FALIURE TO DO SO COULD MEAN ANYONE CAN DELETE
ALL YOUR ARTICLES & OTHER DATA.
