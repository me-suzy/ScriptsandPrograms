Installation
Installing the script is pretty simple. This section will guide you through the steps to have your blog up and running in minutes.

Back to Top

Database configuration
In order to install the script you will need to create a MySQL database and a user which has permissions to access this database.

Once you have these, open up includes/config.php. Here you will need to enter your information. Usually you should be able to leave $dbhost and $dbprefix as default. You don't need to change these unless you know what your doing.

You will however need to change:

$dbuser = "";
$dbpass = "";
$dbname = "";

Enter the name of your database into $dbname such as $dbname = "mydatabase";. You also need to enter the username for the user who can access this database into $dbuser and the password for that user into $dbpass.

Once you have entered these values, save config.php and upload it to your web server. You may want to remove write permissions to it so that you do not overwrite it in future (although make sure it still has read permissions so the script can read it!).

Back to Top

Running the installer
Once you have uploaded the file you can access the installer and run the script:

install/install.php

If you are reading this on the website you will be able to click the above link but if not simply type the address into your web browser. The installer will then take you through each step.

Delete /install/
For security reasons, once you have run the installer you will need to delete the install folder from the web server. Do it. Do it now!

Admin Panel
You can reach the admin panel by goign to admin.php from your homepage. You will then be asked to log in using the username and password you choosen when installing the script. Once you have logged in you will be taken to the configuration page. You can update your password and details by clicking on the my account page.

Back to Top

Configuration
From here you can alter the various configuration settings of the script. Each one is simply a text box so make sure that you enter the correct value. There is an explanation of each one next to each other and usually an example of the default value in case you need to reset it.

Back to Top

Editing Pages
Back on the subject of the regular pages. If you are on a wiki page you can click edit to open the page source in a text box. You can edit the page here and once your done click preview to test it or save page to save the new copy. Also use the text box below to explain what you are changing for the history. An explanation of how to edit is below the text box.

Back to Top

Page History
Each time you save the page a new copy is created and all the other copies are archived. You can go into the history page and click the date to see a copy of that page. You can also revert back to that copy of the page by selecting the radio button and clicking the revert button at the bottom.

Skins
The default skin for Particle Wiki is ParticleBlue. However the script also ships with several other skins. These Are based on the ParticleBlue templates though so ParticleBlue must be still there while using the other skins. Alternatively you can copy all the missing files into the folder. Also feel free to develop your own skins and submit them on the forums.
