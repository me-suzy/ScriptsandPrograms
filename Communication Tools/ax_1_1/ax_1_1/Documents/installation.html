AnnouncementX New Installation Guide
Welcome! This guide will help you to install and configure AnnouncementX.

To install AnnouncementX, please, follow these steps:

1. Upload all the files from the 'upload' directory to a directory you on your server.
This server should have php and mysql installed to run AnnouncementX
2. Chmod files and folders as it is shown in this table
3. Run the installer by using the installation file: www.yoururl.com/yourfolder/install/install.php
4. After the installation you must chmod the conf_global file once again: chmod it to 0666
5. Now you've succesfully installed AnnouncementX and can enjoy using it.

Note: after the installation process the installer will be locked. To unlock it you will have to delete the following file: finish.lock in the install directory

Note: it is not recommended to terminate the installation process before it is finished. In case that happened clean the chosen mysql database or choose another one.

Updating AX from 1.0 to 1.1


So, here's the list of things you have to do to update your AnnouncementX v.1.0 to AnnouncementX v.1.1
  1. Download the newest version (v.1.1)
  2. Upload new files and replace the old ones during the upload (except conf_blobal.php)
  3. Run file update.php from the root directory (where AX's index.php is)
  4. Follow the update process and fill in the required fields
  5. Enjoy using the newest version of AnnouncementX
