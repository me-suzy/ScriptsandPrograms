AnnouncementX Readme

AnnouncementX is a tool that allows you to make your site better. It works as an announcement board that gives your visitors an opportunity to discuss every announcement you make. Also, users, registered at AnnouncementX, can send each other private messages and have their own profiles that are easy to edit.

If it is your first installation, please, read the Installation Guide. This guide will show you how the software should be correctly installed and configured to avoid security problems and site crashes in future.

It's important to admit that there're some requirements for the server that is planned to use for installing AnnouncementX. These requirements are:
PHP: 4.1 or better
MySQL: 4.0.11 or better
Preferred OS: Linux, Unix
*Note: if you install AX on a server that uses Windows, there can be some serious problems, so it means you do it at your own risk.

In case you do not know what is used on your server it's highly recommended to ask your server administrator about the required things.

Note: if you install this software on the sever that has earlier version of php or mysql (or both) it is not guaranteed that AnnouncementX will work properly or will work at all, you do it at your own risk.

Note: in case you've found a bug, you can always report about it by sending an e-mail from our site: www.announcementx.xs3.com
