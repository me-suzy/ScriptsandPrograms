Help

This application allows the user to monitor the evolution of a person or of a teams skills by comparing the levels achieved at regular time interval.

1. With the menu Administration, create or delete the stages, areas or skills as a function of the goals given to a specific team.

 2. Create an evaluation of the skill, the desired goal and then grade the level required to reach each skill (this application is not use to estimate the level reached). The number of stages can be adjusted when the application is created.

 3. At fixed intervals, create a new evaluation and compare the evolution and the progress made vs. the goal that had been given: the grade posted is the average of skills by areas.
