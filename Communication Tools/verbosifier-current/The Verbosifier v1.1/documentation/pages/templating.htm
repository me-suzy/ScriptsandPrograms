Templating

The Verbosifier uses two templates in a way much like a box. As an analogy, shout.htm is how a book looks, and main.htm is the big packing box where all the books go.

The Verbosifier's templates should preferably be written in XHTML 1.0, but HTML 4.01 will work fine too.

shout.htm

How each individual shout will look when displayed.
{NAME}: shouter's name. Will be made into a link to the shouter's website if a valid URL was entered.
{DATE: the date or time the shout was made. The format can be configured in admin/control.php.
{SHOUT}: the text of the shout.

Back to contents

main.htm

How the entire page will look.
{SHOUT}: displays all the shouts as formatted in shout.htm.
{FORM}: displays the form for making a new shout.
