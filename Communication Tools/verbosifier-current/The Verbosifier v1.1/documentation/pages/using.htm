Using The Verbosifier

To use The Verbosifier in your website simply include verbose.php in your code. There are four ways to do this:

Most probably you will have your Shoutbox in an iframe (inline frame). Just create an iframe with the source pointing to verbose.php. For example: 

	<iframe src="verbosifier/verbose.php" frameborder="0" height="100%" width="100%"></iframe>

You can also include it in a normal frameset: 

	<frameset cols = "*, 20%">
	    <frame src ="whatever.htm" />
	    <frame src ="verbosifier/verbose.php" />
	</frameset>
The method I recommend is to simply use PHP's INCLUDE() function: 
	<?php include('verbosifier/verbose.php'); ?>

You can use Server-Side Includes, but SSI is cumbersome and often requires you to rename your pages to .shtml, which isn't possible if you're running a site with dynamic scripts. The SSI to include verbose.php is: 

	<!--#include virtual="verbosifier/verbose.php" -->
