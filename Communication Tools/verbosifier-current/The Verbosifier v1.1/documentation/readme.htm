Installing

  1. Extract the files from the .zip, making sure to preserve the directory structure (clicking the 'Extract to...' button instead of dragging everything to the desktop will do this).
  2. Open admin/control.php with a plain-text editor and enter your details where indicated. You can also change the names of the mySQL table, but it's not necessary.
  3. Upload everything except the documentation/ folder to your server's public area (this is usually where your FTP client puts you by default).
  4. Run install.php in your browser. If all goes well the tables will be created in your mySQL database and you'll be told that you can now run The Verbosifier.
  5. Delete install.php for security.
