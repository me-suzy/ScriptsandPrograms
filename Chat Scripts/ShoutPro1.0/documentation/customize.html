ShoutPro 1.0 - Customizing

How to Do Some Simple Tasks with ShoutPro:

  • Ban a user from the Shoutbox: To ban a user from the shoutbox, first find his or her IP with the instructions below. Then add the IP to the file lists\bannedips.php. Each IP should be on a new line, starting on the second line.
  • View the IP Address of a Poster: In the shouts.php file, each shout is on a new line. The various elements of a shout are stored on each line. Poster's name, message, date, time, and IP are each separated by |^|.
  • Clear the Shoutbox: In your web browser, open the file "admin.php" in your ShoutPro folder. Enter the password you typed into "config.php" and hit submit. A message should appear saying the shoutbox has been cleared.
  • Add More Smilies: Begin by uploading the smilie image file into the "smilies" folder. Then open up "smilies.php" in the "lists" folder in a text-editing program. You can add or remove smilies here. Make sure you follow this format: "SMILIECODE|^|IMAGEFILE|^|". Each unit like that should be on a separate line.
  • Add/Edit Restricted Names: Open the file "lists/names.php" in a text-editing program. Each restricted name and password should be on its own line, starting one Line 4. Do not edit the first three or last two lines of names.php or anyone will be able to view the restricted names and their corresponding passwords over the web. All lines other than those should be in the "NAME|^|PASSWORD|^|EMAIL|^|" format.
  • Add/Edit ShoutCode and Profanity Filter: Open the file "lists/shoutcode.php" or "lists/profanities.php" in a text-editing program. These files are in the same format as the smiles.php file. In shoutcode.php, the first item is the ShoutCode tag and the second is the corresponding HTML tag. In profanities.php, the first item is the profanity and the second item is the filtered word.
