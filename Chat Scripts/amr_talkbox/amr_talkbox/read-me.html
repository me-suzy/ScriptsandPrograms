10 easy steps to installing and setting up your own AMR TalkBox.

1.: Decompress the amr_zip file you just downloaded.
2.: If you don't have the software, click here.
3.: Search for one of the following, stuffit expander or winzip [mac or pc].

4.: Now that you have decompressed the files, let the customizing begin.
5.: Open talkbox.php with a text editor such as TextEdit [mac].
6.: We used Adobe GoLive to edit and write most of the code. Available for Mac or PC.
7.: Once in talkbox.php edit the variables at the top of the script.

Examples

Ex.: $col_text = "#FFFFFF";[Standard text posting color white "#FFFFFF"]
Ex.: $col_text = "#000000";[We changed the standard color to black "000000"]

Ex.: $back_ext = "#6E7B8B";[Background color off-grey "#6E7B8B"]
Ex.: $back_ext = "#6E7B8B";[Background will now show black "#000000"]

Reminder, change only code between the quotation marks.
Do not change anything else unless you know PHP.

8.: Okay your almost done. Now copy and paste the code below into your HTML page.

<iframe frameborder=0 src="http://www.ursite.com/amr_talkbox/talkbox.php" width="430" height="300"></iframe>

9.: Remember to substitute [http://www.ursite.com/amr_talkbox/talkbox.php] with the correct path to your talkbox directories location.

10.: Open your FTP client software and navigate to your domain. Upload the decompressed amr_talkbox . Once finished, choose the talkbox.txt file and set it's permissions to 744 [ owner = rwx, group = r, everybody = r ].

11.: If you are upgrading from an older version of AMR's TalkBox prior to v2.25, we suggest that you replace all files excepet you talkbox.txt file. All your posting will be intact and displayed in the talkbox as usual. Please note that you should make a back-up of all your files before proceeding with any update. Thank you
