Smiley Shortcut Introduction

In the special flash chat client, you can send smileys to the chat room using shortcuts in an MSN messenger similar way.

Thus people can call the smiley to express themselves either clicking the icon
or a special keyboard shortcut if they can remember,
or even edit pre-defined smileys for chat.

There are altogether 24 shortcuts of smiley, showing below:
