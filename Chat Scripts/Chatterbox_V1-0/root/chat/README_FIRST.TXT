Installing this script is very, very easy. 
If you have a PHP enabled server and a MySQL database, read on......

Create a directory on your server to receive the script.
Upload all the files and folders to your webserver. Keep the directory structure intact.
Chmod the chat directory you created to 777.
Chmod the banners directory to 777.
Chmod the file lk.php to 777.
Edit the file db_file.php and add your database settings.
Browse to http://www.yoursite.com/chat_directory/install.php.
Follow the instructions given in the installer.
That's all.
