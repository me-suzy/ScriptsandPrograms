 Setting up the config.php file:  

Before using the calendar on your website, it is necessary to set up the file called config.php. In here are all the settings you will need to set the calendar to match your requirements. Each variable has a comment with it that tells you what to change and what it will affect.

Edit the file using notepad or something similar. Once you have finished editing the file, upload it to your server and observe the effects. If you are unsure about any of the variables, simply change them and see what happens!

Adding the calendar to your web pages:  

Getting the calendar onto your page is easy. If you are using a php page you can just use an include statement like this

<?php include "http://www.yoursite.com/calendar/calendar.php"; ?>

where you would like the calendar to appear. Another method is to use an iframe like this

echo"<iframe src='http://www.yoursite.com/calendar/calendar.php' width='190' height='240' frameborder='0' scrolling='no'></iframe>";

If you are using html pages, call it in an iframe like this

<iframe src=http://www.yoursite.com/calendar/calendar.php width=190 height=240 frameborder=0 scrolling=no></iframe>

You will need to adjust the width and height of the iframe to suit but using an iframe in this manner gives you a good deal of control over the size of the calendar output. You can copy and paste any of the examples above but remember to change the URL's!

 
     
Adding items to the calendar:  

To add an item click the "Add" button. You must give the item a title and description and you must also assign a date to that item where it will show up on the calendar. If you miss out any of these, the script will generate an error message. If you attempt to add an item to a date for which there is already an item assigned, you will receive an error message prompting you to edit that entry rather than trying to add another one.

 
     
Editing items in the calendar:  

If you need to edit an entry, simply click the "edit" button and follow the prompts.

 
     
Deleting items from the calendar:  

To delete an entry from the calendar, click "Edit", find the entry you wish to delete and click the "Delete" button. Note* This is a permanent operation and cannot be undone. Once it's gone, it's gone! You will not be prompted to confirm the deletion. We understand that people are smart enough to know which entry they want to delete and have made a conscious decision to do so.

 
     
Adding multiple items to one date:  

You can make the calendar display multiple entries on a single date by adding them all to one item. For example, if you wanted four items on one date, just add a new item and put all four entries on that one item. Because you can use html in the script, you can use a table or lay the items out in any way you want.

If you have added an item for a particular date and, at a later time want to add more, just go to "Edit" and edit the entry to add the new items.

 
     
Using HTML:  

You are able to use html in the calendar. This means that any html code will work. You can make the calendar show hyperlinks, images or whatever you want. You can link to news items on your sites for instance and make the calendar work as a sort of archive.

 
     
Reserved words:  

There are certain "reserved words" that you cannot use with this script. You cannot use the word "-item-" nor can you use the symbols "|**|" together. The script uses these to differentiate between things like the date and the entry you have made when it stores your items. Whilst using these will not break the script, neither will it be able to display the entry correctly.
