Introduction

Time Tracking by TimeMagnum (http://www.quickscheduling.com/ ) is an easy to install application designed to allow a company or individual the ability to manage and prioritize their schedules online and be assessable from anywhere around the world.  The application features task prioritization and an easy to use interface.  The easy install wizard makes it a quick setup. 

 

 

SQL Table Structure

 

Table: users

Table: userRoles

Table calendar_tasks

 

Default Calendar Admin is injected into SQL Tables at Install Time.

Username: admin

Password: admin

 

 

Critical File Structure

 

calendar_install.php

Should be executed first in order to setup the calendar software and create the MySQL database.

 

calendar_config.php

Is written to at install time.  We highly recommend you chmod or use an applicable application/OS function to allow this file to be written/modified.

 

login.php

Existing users login to their calendar and task list or create a new user account.

 

calendar.php

Displays a list of current tasks and the calendar, requires user to be logged in.

 

taskdetails.php

Allows the users to add a new task to the calendar.

 

usermanage.php

Allows the deletion of users or make a user an administrator.
