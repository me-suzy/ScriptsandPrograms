  1. Firstly unzip all the files into a local directory on your computer.
  2. Upload the whole contents of the "Upload" directory on to a directory on your web server.
  3. Make sure the permissions for the file "inc/config.inc.php" and "inc/settings.inc" is set to 666. This is so that the installation script can write to the file.
  4. In your web browser navigae to the location of your blog with /install/ after it.
  5. This loads up the installation script.
  6. Follow the on screen wizard making sure you fill in the settings correctly.
  7. On completion of the installation wizard please remove the install directory to make sure no one else runs it.
  8. You have completed the installation procedure.
