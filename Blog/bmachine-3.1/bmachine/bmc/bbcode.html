boastMachine BB Code help

[URL] [/URL]
Used to create a link
eg: [url]http://yahoo.com[/url] OR [url=http://site.com] CLICK HERE [/url]

[B] [/B]
Print Bold text
eg: [b] Some text [/b]

[I] [/I]
Print Italicised Text
eg: [i] Some Text [/i]

[U] [/U]
Print underlined Text
eg: [u] Text here [/u]

[S] [/S]
Print StrikeThrough text
eg: [s] Text here [/s]

[COLOR=color] [/COLOR]
Print colored text
eg: [color=red] Colored text [/color]
      [color=#FF00FF] Colored Text [/color]

[SIZE=size] [/SIZE]
Print sized text
eg: [size=4] Sized Text [/size]
      [size=7] Sized text [/size]

[FONT=font] [/font]
Print text is some particular font
eg: [font=Verdana] TEXT Here [/font]
      [font=Comic Sans MS] Text here [/font]

[CODE] [/CODE]
Print text in CODE format

     Code:
     This is a sample Coded text
    eg : [code] This is some coded text [/code]

[QUOTE] [/QUOTE]
Print some text int the quote form

     Quote:
     This is a sample Quoted text
    eg: [quote] Quoted text [/quote]

[img][/img]
Include an image
eg: [img]http://site.com/image.gif[/img]

You can produce bbCode combinations like embedding an [IMG] in a [URL]
eg:
[url=http://site.com][B] Click here [/B][/url]
[url=http://site.com][img]http://site.com/image.png[/img][/url]
